transition-behavior: allow-discrete

Animate display, visibility, and other discrete CSS properties

Baseline 2025 Chrome 117+ Firefox 129+ Safari 17.4+

01 With vs Without allow-discrete

Without transition-behavior: allow-discrete, toggling display: none snaps instantly. With it, the element transitions out before hiding, and transitions in on show.

04 Visibility vs Display Transitions

    

      visibility keeps the element in layout (space preserved). display: none removes it entirely. Both can now be transitioned with allow-discrete.
    

    

      

        
Compare behavior

          Key difference: When visibility transitions, the element fades but its space remains.
          When display transitions, the element is fully removed from layout after the transition ends.
